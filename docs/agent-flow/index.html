      02 / Technical blueprint

      The script stays human-authored.
      Agents build the film around it.

      Agents never own state directly. The input script is content-addressed and immutable; production intelligence lives in sidecar annotations. Every later decision becomes a versioned artifact and every release requires machine evidence.

      If a factual, cultural, rights, or production issue is blocking, the system asks the user for a corrected script revision. It never silently rewrites the supplied text.

      Release logic

      Quality contract graph

      01Hard invariantssource, dignity, anatomy, identity, consent
      02Modality floorsstory, visual, voice, sound, captions
      03Evidence confidenceindependent detectors + calibrated judges
      04Aesthetic rankingcinematic unity, emotion, rhythm, memorability

      A high overall score can never compensate for a failed hard invariant. Rubric v1 is diagnostic; v2 must be calibrated before it can authorize release.

      Every intervention, and what replaces it

      Launch: four intentional decisions
      Human decisionWhat the user doesWhat the agent does nextUX rule
      1 · Submit exact scriptPaste or upload the complete scriptHashes it; adds production annotations in a separate sidecarNo rewrite button and no silent “improvement”
      2 · Pick the lookChoose among 117 looks; Glowing Divine Realism is defaultDerives and locks the reusable Nano Banana style tailNo Recommended tab; tail is inspectable, not another mandatory form
      3 · Lock cast & locationsAccept each AI option, edit its prompt and regenerate, or upload an imagePersists the chosen anchors, then creates character sheets automaticallyThree obvious actions on every asset; no hidden workflow
      4 · Review final videoWatch the finished episode and accept or request a targeted rerunExports the passing revision and packages every editable ingredientThe user is reviewing a film—not operating a pipeline
      Later · Review clipsAccept, reject, edit prompt/references, or regenerate a clip before editRebuilds only affected dependencies and reconforms the editPlanned add-on; deliberately excluded from the launch flow

      Design principle: human taste is concentrated where it compounds—script, world, identity, and final film. The expensive operational middle stays autonomous and remains inspectable rather than becoming a sequence of approval forms.
